# Operating System (운영체제)

**Category:** IT 기초 / 하드웨어·소프트웨어
**Status:** stable
**Last updated:** 2026-04-12

## TL;DR

운영체제(OS)는 하드웨어 자원을 관리하고 응용 프로그램을 실행하는 **메인 프로그램**이다. 네트워크 통신, 입출력 장치 관리 등 컴퓨터 사용에 필요한 모든 것을 총괄하며, 그 핵심에는 프로그램들을 중재하는 **커널(Kernel)**이 있다. 서버 환경에서는 **Linux**가 사실상의 표준이다.

## 설명

### OS의 3대 역할

1. **프로그램 실행 관리** — 응용 프로그램(카톡, 브라우저, Excel 등)을 메모리에 올리고 CPU 시간을 배분한다.
2. **하드웨어 자원 관리** — CPU, 메모리, 디스크 등 하드웨어 자원을 프로그램에 할당하고 회수한다.
3. **네트워크·입출력 관리** — 네트워크 통신, 키보드·마우스·모니터 등 입출력 장치를 제어한다.

### 커널 (Kernel)

커널은 OS의 **핵심 구성요소**로, 프로그램들이 하드웨어 자원을 사용할 때 이를 중재하는 역할을 한다. OS를 만들 때 커널을 기본으로 하고 그 위에 다른 구성요소(셸, 파일 시스템, 드라이버 등)를 올린다.

- 프로그램 간 CPU·메모리 할당을 조율
- 하드웨어와 소프트웨어 사이의 **추상화 계층** 제공
- 인터럽트 처리, 프로세스 스케줄링 담당

### 주요 OS 비교

| OS | 특징 | 주 사용처 |
|---|---|---|
| **Linux** | 오픈소스, 서버 OS 사실상 표준 | 서버, 클라우드, Android 기반 |
| **Windows** | 데스크톱 점유율 1위, GUI 중심 | 개인 PC, 기업 업무 환경 |
| **macOS** | Apple 전용, Unix 기반 | 개발자·디자이너 워크스테이션 |

### Linux가 서버 표준인 이유

- **오픈소스** — 무료, 커스터마이징 자유
- **안정성·보안** — 서버 24/7 운용에 최적화
- **경량** — GUI 없이 CLI만으로 운영 가능, 리소스 절약
- **파생 OS** — Android(모바일), Ubuntu, CentOS, Amazon Linux 등이 모두 Linux 커널 기반

### AI·바이브코딩과의 관련성

바이브코딩 도구(Bolt.new, Replit 등)가 만들어주는 웹 앱을 배포하면, 그 서버는 거의 100% Linux 위에서 돌아간다. Vercel, AWS, GCP 등 모든 클라우드 서비스의 기반이 Linux이기 때문이다.
